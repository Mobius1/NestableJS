NestableJS

Create Drag & Drop Heirarchical Lists.

Introduction

NestableJS is a javascript library for creating drag & drop heirarchical lists.

Features

  • No dependencies.
  • Written in modern ES6 javascript.
  • Lightweight - less than 5kB gzipped!
  • Compatible with modern browsers.
  • Touch enabled for use with mobile devices such as tablets and smartphones.
  • Customizable.
  • Fluent API.
