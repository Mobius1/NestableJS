    Using [data-nestable-*] Attributes

    Nestable can utilise HTML5 [data-nestable-*] attributes for further customisation.

    The attributes must be added to the list item elements.

    <ol>
    <li>Item 1</li>
    <li id="item2">Item 2
        <ol>
            <li data-nestable-parent="item2">Item 3</li>
            <li data-nestable-parent="item2">Item 4
                <ol>
                    <li data-nestable-parent="item2">Item 5</li>
                    <li data-nestable-parent="item2">Item 6</li>
                </ol>					
            </li>
        </ol>
    </li>
    <li data-nestable-axis="x">Item 7</li>
    <li>Item 8</li>
    <li>Item 9
        <ol>
            <li>Item 10</li>
            <li>Item 11</li>
        </ol>
    </li>
    <li data-nestable-axis="y">Item 12</li>
    <li>Item 13</li>
    <li data-nestable-disabled>Item 14</li>
</ol>

    Attribute reference

    Attribute Name Value Description
    [data-nestable-axis] "x" or "y" Confines the list item to either the x (left / right) or y (up / down) axis when moving. Setting to "x" will allow nesting, but prevent reordering. Setting to "y" will allow reordering, but prevent nesting.
    [data-nestable-disabled] Disables the item preventing dragging and nesting within it. The value can be omitted or a boolean can be used. Same as adding the .nst-disabled className to the item.
    [data-nestable-parent] CSS3 ID selector string Confines any list item to the parent defined in the value. The value must be the id string of the parent element WITHOUT the #.
